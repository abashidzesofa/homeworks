1. Как скопировать на свой компьютер репозиторий из github.com, что должно быть сделано предварительно?

- Gitbush на компьютере.
- Аккаунт на Github со сгенерированным SSH ключом.
- Зайти в нужный репозиторий и скопировать ссылку с SSH на нужный нам репозиторий.
- На Gitbush выбрать рабочую папку, куда будем копировать репозиторий.
- git clone и ссылка на скопированнй репозиторий.




2. Как создать git репозиторий из локального каталога на своем компьютере. Как его отправить на github.com?

git add
git commit -m ""
git push



3. Как получить на локальный компьютер изменения с github.com

git pull



4. Какую информацию нам дают команда :
    git status  - показывает состояние файлов в git репозитории.
    git log  - показывает список коммитов, которые в текущий момент есть в репозитории git
    git remote -v   - показывает список удаленных репозиториев, связанных с локальным репощиторием на нашем компьютере.
